    Localization

    By default android widget will use device's language: Swedish if it is chosen as deviceâ€™s default language and English in other cases. On iOS side we use application's language.

    Currently widget supports only English("en") and Swedish("sv"). If you want to force app to use language that differs from current one - you can use methods below:

    You can set localization buy executing setLocale in antourage fab object. 

    antfab.setLocale("en") // or "sv"

    You can set localization buy setting widgetLocale property in antourage view componet. 

    import AntourageView from 'react-native-antourage';

// ...

<AntourageView widgetLocale={'en'} /> // or "sv"
